template<typename... Params>
bool pros::lcd::print ( std::int16_t line,
const char * fmt,
Params... args )

Displays a formatted string on the emulated three-button LCD screen.

This function uses the following values of errno when an error state is reached: ENXIO - The LCD has not been initialized. Call lcd_initialize() first. EINVAL - The line number specified is not in the range [0-7]

Parameters
lineThe line on which to display the text [0-7]
fmtFormat string
...argsOptional list of arguments for the format string
Returns
True if the operation was successful, or false otherwise, setting errno values as specified above.
